## 🚀 Installation

### One-Liner (run without Installation)
```bash
nix run 'github:spitfiregg/nixvim'
```


### Installation
```bash
git clone https://github.com/SpitfireGG/nixvim.git && cd nixvim
nix flake update && nix build .#packages.x86_64-linux.default
./result/bin/nvim
```
